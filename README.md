## Using this example

To run the project :

`./gradlew bootRun`


To create the jar file:

`./gradlew bootJar`


To run lint (currently it is using the default rule set)

`./gradlew detekt`


Configuring the repository to use versioned hooks

`git config --local include.path ../.gitconfig`
